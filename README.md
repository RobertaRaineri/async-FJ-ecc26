# async-FJ-ecc26
Simulations for the paper *"Asynchronous Friedkin-Johnsen Dynamics and Wake-Up Centrality"* by Raineri R., Proskurnikov A. - submitted to European Control Conference 2026. 

### Paper Abstract ###
We study randomized asynchronous opinion dynamics in the Friedkin–Johnsen (FJ) model, where agents update asynchronously according to random activation. We provide a direct convergence proof that establishes an explicit pathwise convergence rate. We analyze how this rate depends on the activation probability vector and prove its monotonicity, demonstrating convexity for certain influence matrix classes. We formulate the rate maximization problem as an optimization with linear-bilinear constraints, enabling efficient numerical solution via sequential linear programming. In the single-agent update setting, the optimal activation distribution defines a novel centrality measure -- "wake-up centrality" -- that quantifies each agent's importance for rapid equilibration. Numerical experiments on random and real networks show that wake-up centrality differs substantially from classical centralities while correlating with influence-based measures. Our results connect opinion dynamics, optimization, and network centrality theory.
